## Usage
- Clone the repository and set working directory to 'speed-typing-game'
- Create a virtual environment and install required dependencies:
    ```
    pip install -r requirements.txt
    ```
- run the application:

    ```
    python3 .\speed_typing_game\main.py
    ```
